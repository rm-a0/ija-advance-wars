## Overview

Simple turn-based strategy game writting in Java.

### How to run

Prerequisites:
- JDK 21
- Maven

Run from project root:

```bash
mvn javafx:run
```

Or build a runnable JAR and run it:

```bash
mvn -DskipTests package
java -jar target/ija-game-1.0-SNAPSHOT.jar
```
